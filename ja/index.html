Misskeyをはじめよう

全てがオープンソースの、効率的で究極に洗練された分散マイクロブログSNS

Misskeyを見つけていただき、ありがとうございます。Misskeyは、地球で生まれた分散マイクロブログSNSです。Fediverse(様々なSNSで構成される宇宙)の中に存在するため、他のSNSと相互に繋がっています。

暫し都会の喧騒から離れて、新しいインターネットにダイブしてみませんか。

Misskeyをはじめる理由

多機能

Misskeyは多機能です。

  • 投稿の文字数制限と公開範囲

    Misskeyでは１投稿につき1000文字まで書くことができます。そして、投稿の公開範囲を段階的に定めることができます。また、迷惑防止のために画像や投稿を予め隠しておくことができます。

  • リアクション

    「😆」「🤔」「👍」などの9個の絵文字から1つを選んで、他のユーザーの投稿にリアクション(反応)することができます。

  • Misskey Flavored Markdown

    投稿やプロフィールでMarkdown風の構文を使うことができます。太字やコード、リンクなどを表現できます。

  • アンケート

    最大10個の選択肢を作成しアンケートをとることができます。

  • ホーム・デッキ (デスクトップ)

    デスクトップでは、幅広い画面の「ホーム」とカラム型レイアウトの「デッキ」とを使うことができます。

  • ウィジェット

    通知、アクティビティ、カレンダー、トレンドなどを、自由に並び替えられるウィジェットとして利用できます。

  • ドライブ

    投稿された画像はドライブに保存されます。ファイルをドライブに直接アップロードすることもできます。

  • ゲーム

    他のユーザーと対戦・交流できるゲームが遊べます。
    「リバーシ」はユーザーのアイコンが石として表示されるほか変則ルールでも遊ぶことができます。

独立性と社会性

あなたの意志で運営者を選択し、もしくはあなた自身でサーバーを運営することもできます。

  • Fediverse

    Misskeyでは、他のサーバーのユーザーをフォローすることができます。ActivityPubという規格のサーバー間通信機能を持っているためです。
    そのような機能を持つSNSサーバーどうしの連合体は、無数の星が輝く宇宙のように見たてて「Fediverse [Federation(連合) + Universe(宇宙) の造語]」と呼ばれます。 Fediverseを形成するソフトウェアはMisskeyだけではありません。

  • インストールが簡単

    Misskeyサーバーの構築は簡単です。
    → セットアップ方法

オープン

Misskeyはオープンソースソフトウェアです。開発や翻訳に参加することができます。独自の拡張も追加することができます。

GitHubCrowdin(翻訳)

  • API

    MisskeyのほぼすべてのAPIエンドポイントを、アプリを作成して利用することができます。Misskeyの機能を活用したBotやクライアントを作ることができます。
    → Misskey API

  • AGPLv3

    MisskeyはGNU Affero General Public License v3.0 (AGPLv3)の下で二次利用することができます。
